# My Expo Project

## Introduction

Welcome to My Expo Project! This is a React Native application built using Expo. The project aims to provide a seamless and efficient mobile application development experience.

## Prerequisites

Before you begin, ensure you have met the following requirements:

- **Node.js**: Install the latest LTS version of Node.js from [Node.js](https://nodejs.org/).
- **npm**: Node.js package manager, which comes with Node.js.
- **Expo CLI**: Install Expo CLI locally in your project.

## Installation

To install and set up the project, follow these steps:

1. **Go to project**:
    ```sh
    cd TECHNICALTEST
    ```

2. **Install dependencies**:
    ```sh
    npm install
    ```

## Running the Application

To start the development server and run the application, use the following command:

```sh
npx expo start
```
